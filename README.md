# Speedtyper

A minimalist, distraction-free speed typing practice app — zero dependencies, zero build step.

## Features

- **Instant play**: Open `index.html` in any browser and start typing.
- **Real-time stats**: Live WPM and accuracy tracking.
- **Visual feedback**: Correct characters turn white; errors highlighted in red.
- **Curated text library**: Pangrams, quotes, and programming wisdom for varied practice.

## Usage

1. Open `index.html` in a modern browser.
2. Start typing — the app tracks your progress automatically.
3. Press **Space** or **Enter** after completing a segment to start a new round.

## Tech

Vanilla HTML, CSS, and JavaScript. No frameworks, no build tools, no server required.
